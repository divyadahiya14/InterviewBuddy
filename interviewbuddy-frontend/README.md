# Interview Buddy 🚀

Interview Buddy is a modern, premium AI-powered mock interview platform designed to help candidates level up their technical skills. 

## ✨ Key Features
- **AI Interviews**: Practice DSA, System Design, and Fundamentals with real-time AI feedback.
- **Expert Human Sessions**: Book 1-on-1 mock interviews with industry experts.
- **Comprehensive Feedback**: Get detailed analysis of your performance, including code quality, time complexity, and more.
- **Professional Dashboard**: Track your progress and manage your interview schedule.

## 🛠️ Technology Stack
- **Frontend**: React, Vite, Framer Motion, Monaco Editor
- **Backend**: Spring Boot, MySQL, OpenAI/OpenRouter API
- **Styling**: Modern dark-themed CSS design system

## 🚀 Getting Started

### Frontend
1. Navigate to the frontend directory: `cd MockInterviewX_frontend-main`
2. Install dependencies: `npm install`
3. Start the dev server: `npm start`

### Backend
1. Navigate to the backend directory: `cd MockInterviewX_backend-main`
2. Configure `application.properties` with your database and API credentials.
3. Run the application: `./mvnw spring-boot:run`
